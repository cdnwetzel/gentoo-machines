# Dell XPS 15 9510 — One-Shot Install Pre-Flight Checklist
# Goal: ONE chroot session. Everything works on first boot. No re-entry.
#
# Lessons from: MBP 2015 (6+ re-entries), XPS 9510 v1 (3 re-entries),
#               Surface Pro 6 (1 re-entry target)

## Pre-Chroot (Part 1 + Part 2 — from Live USB)

### Disk Layout (Dual NVMe)

| Disk | Partition | Size | FS | Mount | Purpose |
|------|-----------|------|----|-------|---------|
| nvme0n1 | nvme0n1p1 | 512M | FAT32 | /boot/efi | EFI System Partition |
| nvme0n1 | nvme0n1p2 | ~930G | ext4 | / | Root (Gentoo OS) |
| nvme1n1 | nvme1n1p1 | ~931G | ext4 | /data | Data, ccache, repos |

No swap partition — zram-init handles swap (8GB compressed zstd in RAM).
No separate /boot — kernel files go in /boot on root partition.
No /home split — root has 930GB, plenty of space.
Portage tmpfs: 24GB from fstab (large packages fall back to disk).

### Pre-Chroot Checklist
- [ ] Boot from Live USB
- [ ] Verify both NVMe disks: `lsblk` shows nvme0n1 + nvme1n1
- [ ] Verify WiFi connected (NetworkManager)
- [ ] Run `gentoo_install_part1.sh` (partition + format + mount both disks)
- [ ] Verify mounts: `df -h` shows /mnt/gentoo, /mnt/gentoo/boot/efi, /mnt/gentoo/data
- [ ] Run `gentoo_install_part2.sh` (stage3 + config copy + pseudo-fs mount)

## In-Chroot — The One-Shot Session

### Phase 1: Bootstrap
- emerge-webrsync + emerge --sync
- Set profile: `default/linux/amd64/23.0`
- `emerge --update --deep --newuse @world` (apply USE flags)

### Phase 2: Kernel + Firmware
- Install gentoo-sources, linux-firmware, intel-microcode, installkernel
- `eselect kernel set 1`
- Run kernel_config.sh (26-phase programmatic config)
- `make olddefconfig` + optional menuconfig review
- `make -j17 && make modules_install && make install`
- Verify: `ls /boot/vmlinuz-*`

### Phase 3: Bootloader (GRUB)
- Install GRUB, copy /etc/default/grub (i915.enable_guc=3 + NVIDIA params)
- `grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=Gentoo`
- `grub-mkconfig -o /boot/grub/grub.cfg`
- Verify: `grep menuentry /boot/grub/grub.cfg`
- Verify: `grep enable_guc /boot/grub/grub.cfg`

### Phase 4: System Configuration
- Timezone: America/New_York
- Locale: en_US.UTF-8
- Hostname: xps-9510
- Root password
- Create user `chris` (wheel, audio, video, usb, input, plugdev)
- Sudo: `%wheel ALL=(ALL) ALL`

### Phase 5: Networking (CRITICAL)
- Install wpa_supplicant, NetworkManager, dhcpcd, nm-applet, wireless-regdb
- Verify wpa_supplicant has dbus USE flag
- `rc-update add NetworkManager default`

### Phase 6: All Packages + NVIDIA
- Copy world file to /var/lib/portage/world
- Copy package.use, package.accept_keywords
- `emerge --update --deep --newuse @world`
- This installs: XFCE, LightDM, PipeWire, nvidia-drivers, browsers, dev tools

### Phase 7: Portage Infrastructure
- Create /var/tmp/portage (chown portage:portage)
- Create /var/tmp/portage-disk (chown portage:portage)
- Set up ccache: `/data/build-cache/ccache` (chown root:portage, chmod 2775)
- Verify package.env + env/notmpfs.conf

### Phase 8: OpenRC Services
Default runlevel:
- dbus, NetworkManager, acpid, bluetooth, thermald, tlp
- display-manager, sshd, metalog, local, netmount

Boot runlevel:
- elogind, zram-init, alsasound

### Phase 9: LightDM + XFCE
- Verify: `ls /usr/share/xsessions/xfce.desktop`
- Install lightdm.conf (user-session=xfce, session-wrapper=/etc/lightdm/Xsession)
- Set DISPLAYMANAGER="lightdm" in /etc/conf.d/display-manager

### Phase 10: PipeWire Audio
- Verify: PipeWire + WirePlumber + xfce4-pulseaudio-plugin installed
- Autostart via gentoo-pipewire-launcher (set up after first login)

### Phase 11: NVIDIA + Machine Hardware
- Install 99-module-rebuild.install to /etc/kernel/postinst.d/
- Install prime-run to /usr/local/bin/
- Create /etc/modprobe.d/nvidia.conf (modeset=1, DynamicPowerManagement=0x02)
- Install sysctl-performance.conf to /etc/sysctl.d/
- Configure zram-init (8GB zstd)
- Install tlp.conf (performance on AC)
- Verify firmware: iwlwifi-QuZ-*, i915/tgl_*, intel/ibt-20-*

### Phase 12: fstab
- Already generated by part2 with actual UUIDs
- Verify: 3 UUID entries + portage tmpfs + /tmp tmpfs

## Post-First-Boot

### Immediate Verification
```bash
# WiFi
nmtui
ping -c3 gentoo.org

# NVIDIA
nvidia-smi           # RTX 3050 Ti, driver 590.48

# Display
xrandr               # should show 3456x2160

# Audio
pactl info | grep "Server Name"   # PipeWire
speaker-test -c 2 -t wav

# zram
swapon --show        # zram0, 8G, zstd

# Thermal
sensors              # coretemp values

# Bluetooth
bluetoothctl show

# Services
rc-status default
```

### Restore Desktop
```bash
cd ~/gentoo_dell_xps9315   # or /data/gentoo_dell_xps9315
bash shared/restore-desktop.sh
sudo bash shared/restore-system.sh
```
